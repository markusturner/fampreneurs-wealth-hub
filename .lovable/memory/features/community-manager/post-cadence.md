Cadence per template (community-manager-daily-post edge function):

- student_of_the_month (wins) — monthly
- new_plays_gems (gems) — daily
- welcome_new_members (discussion) — as needed (only when new members exist)
- new_member_orientation (discussion) — as needed (replies to welcome_new_members)
- recommendations (discussion) — daily
- community_wins (wins) — as needed (only when members complete all trust submissions)
- new_updates (updates) — as needed (only when new courses/meetings exist)
